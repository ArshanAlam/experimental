# Sum Lists
### Cracking the Coding Interview 6th Ed.

Chapter 2: Linked Lists

Interview Question 2.6

Determine if a linked list is a palindrome.

Assumptions:
 * Linked List consist of single digit data
 * Data is in a doubly linked list


#### Example:

**Input:**  2 <-> 3 <-> 2
**Output:** true

**Input:** 1 <-> 2 <-> 2 <-> 1
**Output:** true

**Input:** 4 <-> 2 <-> 2
**Output:** false

**Input:** 1 <-> 2 <-> 3 <-> 1
**Output:** false
